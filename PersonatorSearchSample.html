Column Responses
  • Move Date - Returns the most recent move date
  • Phone - Returns the phone number
  • Email - Returns the email
  • Date of Death - Returns the date of death
  • Date of Birth - Returns the date of birth






Options
  • Search Conditions - Specifies the type of search conditions to use when returning records. Only the populated input fields will be compared. A strict search condition is used by default
  • Max Email - Sets the maximum number of emails that can be returned. Default maximum is "3" if this option is not set
  • Search Type - Allows you to specify the type of search to perform
  • Max Phone - Sets the maximum number of phone numbers that can be returned. Default maximum is "3" if this option is not set
  • Sort By - Allows sorting records in a particular order
  • Return All Pages - If enabled, will return all records up to a limit of 500
  • Page - Specifies which "page" to return for a given request. Your "page" length is set by the Records Per Page option
  • Records Per Page - Specifies how many records are in a "page". The default value is 5
  • Age Range - Sets the range of ages returned. Default will return all ages
